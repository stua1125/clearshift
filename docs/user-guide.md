# ClearShift 사용자 가이드

---

## 시작하기

### 로그인
1. 앱을 열면 로그인 화면이 표시됩니다
2. **Google 계정으로 로그인** 버튼을 탭합니다
3. Google 계정을 선택하여 인증합니다

### 회원가입 (첫 로그인 시)
1. Google 인증 후 회원가입 화면이 나타납니다
2. **이름**을 입력합니다
3. 소속 **지점**을 선택합니다 (예: 삼성 강북점)
4. **가입하기**를 탭하면 완료됩니다

## 공유 캘린더 (모든 구성원 공통)

로그인 후 메인 화면입니다. 같은 지점의 모든 구성원이 동일한 캘린더를 볼 수 있습니다.

### 월간 뷰

지점 전체의 근무 현황을 한눈에 파악할 수 있습니다.

- **월 이동**: `<` `>` 버튼으로 이전/다음 달 이동
- **각 날짜 셀에 표시되는 정보**:
  - 근무타입별 인원 수 (예: D:3 N:2 OFF:1)
  - 등록된 이벤트 (색상 dot + 제목)
- **날짜 탭**: 해당 날짜의 상세 정보 (인원별 근무 배정 + 이벤트)

### 주간 뷰

인원별 근무 배정을 상세하게 확인할 수 있습니다.

- **월간 ↔ 주간 전환**: 상단 토글 버튼
- **주 이동**: `<` `>` 버튼으로 이전/다음 주 이동
- **표시 형식**:
  - 세로: 팀원 이름
  - 가로: 7일 (월~일)
  - 각 셀: 근무타입 뱃지 (D, N, OFF 등)
  - 하단: 해당 주의 이벤트 바

## 내 스케줄 (근무 신청)

하단 탭에서 **내 스케줄**을 탭하여 이동합니다.

### 근무 배정하기 (Paint Mode)

**Step 1. Paint Mode 켜기**
- 캘린더 아래 **Paint ON** 버튼을 탭합니다

**Step 2. 근무 타입 선택**

| 표시 | 이름 | 설명 |
|------|------|------|
| D | 주간 | 주간 근무 |
| N | 야간 | 야간 근무 |
| OFF | 휴무 | 정기 휴무 |
| V | 휴가 | 연차/휴가 |
| T | 교육 | 교육/연수 |

**Step 3. 날짜에 배정하기**
- 날짜를 탭하면 선택한 근무 타입이 배정됩니다
- 같은 타입 날짜를 다시 탭하면 **해제** (토글)
- 다른 타입이 배정된 날짜를 탭하면 **교체**

**Step 4. 배정 삭제하기**
- **지우기**를 선택 후 삭제할 날짜를 탭합니다

**Step 5. 제출하기**
- 모든 날짜에 근무를 배정하면 (100%) **제출하기** 버튼이 활성화
- 제출하면 즉시 공유 캘린더에 반영됩니다

### 제출 후 수정
- 제출 후에도 수정이 가능합니다
- Paint Mode를 켜면 자동으로 작성 중 상태로 전환
- 수정 완료 후 다시 제출하면 공유 캘린더에 반영됩니다

## 매니저 전용 기능

매니저는 하단 탭의 **설정**에서 추가 관리 기능을 사용할 수 있습니다.

### 근무타입 관리

**추가**: 우측 하단 **+** 버튼 → 이름, 약어(1~3자), 색상, 카테고리 입력 → 저장
**수정**: 목록에서 편집 아이콘 탭
**삭제**: 목록에서 삭제 아이콘 탭
**순서 변경**: 항목을 길게 눌러 드래그

### 휴가 MAX 관리

하루 동시 휴가 가능 최대 인원을 설정합니다.

- **기본 MAX**: `−` `+` 버튼으로 조절
- **특정 날짜 Override**: 날짜 선택 → 최대 인원 입력 → 추가

### 이벤트 관리

공유 캘린더에 표시되는 이벤트를 관리합니다.

- **추가**: **+** 버튼 → 제목, 기간, 색상, 메모 입력
- **수정**: 이벤트 탭 → 내용 수정
- **삭제**: 삭제 아이콘 탭

## 자주 묻는 질문

**Q. 제출 후 수정할 수 있나요?**
- 네. 언제든 수정 가능합니다. 수정 시 자동으로 작성 중 상태로 전환되며, 재제출이 필요합니다.

**Q. 제출하면 바로 공유 캘린더에 보이나요?**
- 네. 별도 승인 없이 즉시 반영됩니다.

**Q. 지점을 변경하고 싶어요**
- 관리자(Admin)에게 문의하여 소속 지점을 변경할 수 있습니다
