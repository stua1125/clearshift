# ClearShift 사용자 가이드

---

## 시작하기

### 로그인
1. 앱을 열면 로그인 화면이 표시됩니다
2. **Google 계정으로 로그인** 버튼을 탭합니다
3. Google 계정을 선택하여 인증합니다

### 회원가입 (첫 로그인 시)
1. Google 인증 후 회원가입 화면이 나타납니다
2. **이름**을 입력합니다
3. 소속 **지점**을 선택합니다 (예: 삼성 강북점)
4. **가입하기**를 탭하면 완료됩니다

## 근무자 (Worker) 가이드

### 월별 캘린더

메인 화면에서 이번 달 캘린더를 확인할 수 있습니다.

- **월 이동**: `<` `>` 버튼으로 이전/다음 달로 이동
- **오늘 날짜**: 파란색 테두리로 표시
- **일요일**: 빨간색, **토요일**: 파란색 날짜 표시

### 근무 배정하기 (Paint Mode)

근무 스케줄을 등록하는 핵심 기능입니다.

**Step 1. Paint Mode 켜기**
- 캘린더 아래 **Paint ON** 버튼을 탭합니다
- 버튼이 파란색으로 변하면 활성화된 상태입니다

**Step 2. 근무 타입 선택**

| 표시 | 이름 | 설명 |
|------|------|------|
| D | 주간 | 주간 근무 |
| N | 야간 | 야간 근무 |
| OFF | 휴무 | 정기 휴무 |
| V | 휴가 | 연차/휴가 |
| T | 교육 | 교육/연수 |

원하는 근무 타입 버튼을 탭하여 선택합니다.

**Step 3. 날짜에 배정하기**
- 캘린더의 날짜를 탭하면 선택한 근무 타입이 배정됩니다
- 이미 같은 타입이 배정된 날짜를 다시 탭하면 **해제**됩니다
- 다른 타입이 배정된 날짜를 탭하면 **교체**됩니다

**Step 4. 배정 삭제하기**
- 근무 타입 목록에서 **지우기**를 선택합니다
- 삭제하고 싶은 날짜를 탭합니다

**Step 5. 제출하기**
- 화면 하단 진행률 바를 확인합니다
- 모든 날짜에 근무를 배정하면 (100%) **제출하기** 버튼이 활성화됩니다
- 제출하기를 탭하면 매니저에게 전달됩니다
- 제출 후에는 "제출 완료"로 표시됩니다

### 제출 후

| 상태 | 설명 |
|------|------|
| 작성 중 | 아직 제출하지 않은 상태 |
| 제출 완료 | 매니저 검토 대기 중 |
| 승인됨 | 매니저가 승인한 상태 (수정 불가) |
| 반려됨 | 매니저가 반려 → 다시 수정하여 재제출 가능 |

## 매니저 (Manager) 가이드

### 월별 캘린더

팀 전체 근무 현황을 한눈에 파악할 수 있습니다.

- **캘린더**: 날짜별 이벤트와 휴가 현황 표시
- **휴가 현황**: 각 날짜에 `{현재}/{최대}` 형태로 표시
- **팀원 목록**: 화면 하단에 팀원 리스트와 제출 상태가 표시됩니다

### 날짜별 상세 보기

캘린더의 날짜를 탭하면 해당 날짜의 팀원별 근무 배정을 확인할 수 있습니다.

### 주별 캘린더 보기

월별/주별 탭을 전환하여 주간 뷰로 확인할 수 있습니다.
- 가로 테이블: 행 = 팀원, 열 = 요일
- 각 셀에 근무 타입 뱃지 표시
- 좌우 스와이프로 주 이동

### 팀원 스케줄 상세

팀원 목록에서 팀원을 탭하면 개인 스케줄 상세 화면으로 이동합니다.

- **캘린더**: 해당 팀원의 월별 근무 배정 조회
- **근무 통계**: 근무 타입별 일수 요약
- **승인/반려**: 하단 버튼으로 스케줄 승인 또는 반려

### 근무타입 관리

매니저는 지점에서 사용할 근무 타입을 관리할 수 있습니다.

**근무타입 추가**
1. 우측 하단 **+** 버튼을 탭합니다
2. 이름, 약어(1~3자), 색상, 카테고리를 입력합니다
3. **저장**을 탭합니다

**근무타입 수정**
- 목록에서 근무타입의 편집 아이콘을 탭합니다

**근무타입 삭제**
- 목록에서 삭제 아이콘을 탭합니다

**순서 변경**
- 목록 항목을 길게 눌러 드래그하여 순서를 변경합니다

### 휴가 MAX 관리

하루에 동시 휴가가 가능한 최대 인원을 설정합니다.

**기본 MAX 설정**
- `−` `+` 버튼으로 기본 최대 인원을 조절합니다

**특정 날짜 Override**
- 특정 날짜에 다른 최대값을 적용하려면:
  1. 날짜를 선택합니다
  2. 최대 인원을 입력합니다
  3. **추가**를 탭합니다
- 추가된 Override는 amber 색상 칩으로 표시됩니다
- 칩의 × 버튼으로 삭제할 수 있습니다

### 이벤트 관리

캘린더에 표시되는 팀 이벤트를 관리합니다.

**이벤트 추가**
1. 우측 하단 **+** 버튼을 탭합니다
2. 제목, 시작일, 종료일, 색상, 메모를 입력합니다
3. **저장**을 탭합니다

**이벤트 수정**
- 이벤트 목록에서 해당 이벤트를 탭합니다
- 내용을 수정하고 저장합니다

**이벤트 삭제**
- 이벤트 목록에서 삭제 아이콘을 탭합니다

## 자주 묻는 질문

**Q. 제출 후 수정할 수 있나요?**
- 매니저가 아직 검토하지 않은 상태라면 수정할 수 있습니다
- 승인된 스케줄은 수정할 수 없습니다
- 반려된 스케줄은 수정 후 재제출할 수 있습니다

**Q. 휴가 MAX를 초과하면 어떻게 되나요?**
- 해당 날짜에 이미 최대 인원이 휴가를 배정한 경우, 캘린더에 `{현재}/{최대}` 형태로 경고가 표시됩니다

**Q. 지점을 변경하고 싶어요**
- 관리자(Admin)에게 문의하여 소속 지점을 변경할 수 있습니다
