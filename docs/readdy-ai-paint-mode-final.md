핵심 사용 흐름 (이 흐름이 디자인에서 자연스럽게 읽혀야 함):

1. 화면 진입 → 빈 캘린더 + 비활성 Toolbar 확인 (화면 A)
2. "Paint OFF" 탭 → "Paint ON" 파란색으로 전환
3. D(주간) 버튼 탭 → 해당 버튼 강조
4. 캘린더 날짜 셀 탭 → 셀에 파란 배경 + "D" 뱃지 즉시 표시
5. 다른 날짜 계속 탭 → 프로그레스 바 실시간 증가 (화면 B)
6. N(야간) 버튼으로 전환 → N 버튼 강조, 이후 탭하면 보라색 셀
7. 모든 날짜 채움 → 프로그레스 100% 초록, 제출 버튼 활성화
8. "제출하기" 탭 → "제출 완료" 뱃지 표시 (화면 C)

시각 피드백:
- 셀 탭 시 0.95 → 1.0 scale 바운스 (100ms)
- 배정 셀: 근무타입 배경색 30% opacity + 우상단 pill 뱃지
- 프로그레스 색상: 주황(0~49%) → 파랑(50~99%) → 초록(100%)
